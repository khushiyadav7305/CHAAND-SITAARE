# ✨ CHAND SITAARE — Where Code Meets Couture 🌙

**Chand Sitaare** is a premium full-stack MERN application designed for a bespoke fashion experience.  
It blends **technology, creativity, and storytelling** to deliver a seamless and elegant user journey.

---

## 🌌 About the Project

Chand Sitaare is more than just an e-commerce platform — it's a **design experience**.  
From exploring curated collections to experimenting with styles using the **MixMatch Tool**, every interaction is crafted with elegance.

---

## 🚀 Core Features

- 🔐 Secure Authentication (JWT-based Login/Signup)
- 🎨 Designer’s Lab (Inspiration behind each design)
- ✨ MixMatch Tool (Create and explore outfit combinations)
- 🛒 Dynamic Cart & Wishlist (Real-time updates)
- 📱 Fully Responsive UI (Mobile, Tablet, Desktop)

---

## 🛠️ Tech Stack

### 💻 Frontend
- React.js (Vite)
- Tailwind CSS
- React Router DOM
- Lucide Icons

### ⚙️ Backend
- Node.js
- Express.js

### 🗄️ Database
- MongoDB Atlas
- Mongoose

### 🔗 Other Tools
- Axios
- CORS
- JWT Authentication
